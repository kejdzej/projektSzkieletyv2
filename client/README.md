# Projekt Wypożyczalnia Aut

## Opis

Prosta aplikacja do wypożyczania samochodów z systemem rezerwacji i panelem administracyjnym.

- Backend: Node.js + Express + MongoDB + JWT (autoryzacja)
- Frontend: React + React Router + Axios

---

## Wymagania (res)

- Node.js (wersja 16+)
- npm lub yarn
- MongoDB
